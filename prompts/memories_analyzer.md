# 共同记忆分析 Prompt

## 任务

你将收到 **{name}** 的原材料（聊天记录、照片元数据、社交媒体等）。
从中提取你们的共同记忆与关系动态，用于构建 Memories Skill。

**原则：只提取有据可查的内容，不推断。没有依据就标注「原材料不足」。对人物细节保持客观与尊重。**

---

## 提取维度

### 1. 关系时间线

从原材料中识别关键节点：
- 重要日期（第一次见面、重要约定、里程碑事件）
- 关系里程碑（第一次合作、一起旅行、重要时刻）
- 转折点（关系变化的节点，关系疏远/加深、争吵后和解、分离）
- 频繁提到的时间段或季节（「那年冬天」「大三的夏天」）

> **前任模式额外提取**：在一起纪念日、分手日期、复合节点

```
输出格式：
重要日期：[日期 + 事件描述]
里程碑：[列表]
转折点：[列表]
```

### 2. 共同日常与仪式

从日常聊天中提取你们的专属习惯：
- 固定活动（常见的互动形式：一起玩游戏、周末约饭、每天打招呼）
- 共同爱好（一起看的剧、一起讨论的话题、共同的兴趣）
- 专属仪式（固定的打招呼方式、特定节日的互动习惯）
- 只有你们懂的东西（inside jokes、专属昵称、暗号、梗）

```
输出格式：
日常活动：[描述]
共同爱好：[列表]
专属仪式：[描述]
专属语言：[pet names、inside jokes 列表]
```

### 3. 他/她的偏好与习惯

从聊天和互动中提取：

**吃**：
- 喜欢吃什么 / 不吃什么
- 常去的餐厅 / 外卖偏好
- 特殊饮食习惯（奶茶口味、咖啡偏好）

**玩**：
- 喜欢去的地方
- 旅行风格（计划型/随性、喜欢自然/城市）
- 娱乐偏好（电影类型、音乐口味、游戏偏好）

**工作/学习**：（如有相关内容）
- 工作节奏（早鸟/夜猫、高效集中/随时联系）
- 擅长的领域和方式

**表达习惯**：
- 主动联系的频率和方式
- 表达关心的方式（发东西/直接说/行动表达）
- 对礼物/仪式感的态度

```
输出格式：
吃：[描述]
玩：[描述]
工作/学习：[描述，若无则省略]
表达习惯：[描述]
```

### 4. 关系互动模式

从互动记录中提取你们的相处方式：
- 互动节奏（高频日常联系 vs 偶尔深度交流）
- 常见共同话题（聊什么最多、什么话题最带劲）
- 他/她发起互动的特征（什么时候主动找你、怎么找）
- 你们之间的摩擦模式（什么容易产生分歧、怎么化解）

> **前任模式额外提取**：
> - 常见争吵话题和导火索
> - 冷战与修复方式（谁先道歉、怎么和好的信号）
> - 反复出现的未解决矛盾

```
输出格式：
互动节奏：[描述]
常见话题：[列表]
主动联系特征：[描述]
分歧与化解：[描述]
```

### 5. 情感动态

从互动模式中提取：
- 他/她开心时的表现（怎么表达开心、什么让他/她最高兴）
- 他/她不开心时的表现（怎么表达、需要什么样的回应）
- 他/她在意的事情（什么会让他/她认真起来或有情绪）
- 他/她的情感需求（最需要被满足的是什么）

```
输出格式：
开心时：[描述 + 原文示例]
不开心时：[描述 + 原文示例]
在意的事：[描述 + 原文示例]
核心需求：[描述]
```

---

## 输出要求

- 语言：中文
- 没有信息的维度：标注 `（原材料不足，建议追加聊天记录）`
- 有原文依据的结论：加引号标注原话
- 输出结果直接用于生成 memories.md，要求具体有画面感，不要写「可能」「大概」这类模糊表述
- 对人物和关系细节保持温柔与客观，不评价对错
